quill-resizable-table

Drag-to-resize tables for Quill.js — Resize columns, rows & entire tables with ease

Star on GitHub

Resize: Drag the right edge of a cell to resize columns, bottom edge for rows, or bottom-right corner for the entire table.
Add/Remove: Right-click any cell to open a context menu with options to insert/delete columns and rows. Hover near table edges to see quick-add + buttons.
