# Contributing to quill-resizable-table

Thanks for your interest in contributing\! Here is how you can help.

## Getting Started

1. Fork the repository on GitHub
2. Clone your fork locally
3. Create a new branch for your feature or fix

## Development Setup

bash
git clone https://github.com/abhijeetsatpute/quill-resizable-table.git
cd quill-resizable-table
npm install


### Available Scripts

| Command | Description |
|---|---|
| npm run dev | Start the Vite dev server with the demo page |
| npm test | Run the Jest test suite |
| npm run build | Build ESM, CJS, and UMD bundles to dist |

## Pull Requests

1. Fork the repo and create your branch from main
2. Write tests for any new functionality
3. Run npm test to make sure nothing is broken
4. Run npm run build to verify it compiles cleanly
5. Open a pull request with a clear description of what you changed and why

## Reporting Bugs

Please use GitHub Issues to report bugs.

When filing an issue, include:

- Quill version (e.g., 2.0.0)
- Browser and version (e.g., Chrome 120)
- Steps to reproduce the issue
- Expected behavior vs. actual behavior
- Screenshots or screen recordings if applicable

## Code Style

- Written in TypeScript - follow the patterns in existing code
- Keep changes focused and minimal
- Add JSDoc comments for public methods
- Test new features with Jest (JSDOM environment)
